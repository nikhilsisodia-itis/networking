# TCP-IP Network Access Layer

## Introduction to the Network Access Layers

- The Network Access Layer is the lowest layer in the TCP/IP model.
- It is responsible for how data is physically sent over the network (like cables, Wi-Fi, etc.).
- It deals with the hardware part of networking, such as network cards and switches.
- It defines how bits are turned into signals and sent across the network.

## Protocols used by the Network Access Layer

- Ethernet: The most common protocol for wired local area networks (LANs).
- Wi-Fi: Used for wireless networks.
- PPP (Point-to-Point Protocol): Used for direct connections between two devices, like dial-up.
- Frame Relay, ATM: Used in some older or specialized networks.
