# TCP-IP Internet Layer

## Introduction to the Internet Layer

- The Internet layer is responsible for routing data packets across networks.
- It determines the best path for data to travel from the source to the destination.
- The Internet layer handles logical addressing and packet forwarding.
- It assigns IP addresses to devices.

## Protocols Used by the Internet Layer

- **IP (Internet Protocol):**
  - This is the main protocol.
  - It uses IP addresses to identify devices and helps send data packets from one device to another across networks.
- **ICMP (Internet Control Message Protocol):**
  - Used for sending error messages and network information.
  - For example, it helps tools like ping check if a device is reachable.
- **ARP (Address Resolution Protocol):**
  - Helps find the physical (MAC) address of a device when you only know its IP address, so data can be delivered on a local network.
- **RARP (Reverse Address Resolution Protocol):**
  - The opposite of ARP; it helps a device find out its IP address if it only knows its MAC address (rarely used today).
